# Installation

1. Open **Extensions** sidebar panel in VS Code. `View â†’ Extensions`
2. Search for `Krb_Blue` - find the one by **KUSHAL SAHA**
3. Click **Install** to install it.
4. Code > Preferences > Color Theme > **Krb_Blue**
